Esta carpeta contiene el trabajo relacionado con la teoria de
de biomecanica del brazo. Esto corresponde a background necesario para 
el proyecto apoyo en la prevencion de lesiones de muñeca por acciones 
repetitivas.

El problema se aborda usando IMUs para medir el movimiento de un brazo
en el tiempo. con estos datos se recontruyen los movimientos y acciones
realizadas por un paciente. A partir de esta informacion un fisio
terapeuta puede diagnosticar acciones repetitivas peligrosas y 
tomar las acciones correctivas que considere necesarias.

En esta carpeta de matlab, el trabajo se limita a la biomecanica
del brazo.
